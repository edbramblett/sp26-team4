## Description 
Our app will be a place for pet owners to be quickly connected to pet sitters and pet walkers for any of their needs. The app will include the ability to make bookings, a review system, GPS walking routes. Our app aims to create a easy way for pet owners to find care they need, from a wide range of pet caretakers.

## App Functions
1. Customer (the user with the customer role):
    1. Create/modify customer & pet profile - <We will have a profile page, similar to other websites, where you can edit the information for each profile>
    2. Search for available services - <There will be a search menu, with a filters tab to let users find a specific service>
    3. Create bookings with pet caretakers - <After find a service, the user will be taken to a page to confirm a date and time>
    4. Write reviews for services - <After a service is complete, the user will be given a link to a review page to write a review>
    5. Make payments - <When making a booking, the user will pay for the services, and after the serivce will have the ability to leave an additonal tip>
2. Provider (the user with the provider role):
    1. Create/modify/remove provider profile - <Allow caretakers to create their profiles, list their provide services, and days they work>
    2. Connect pet caretakers to customers - <Provice a chat service or similar way for caretaekrs to contact owners>
    2. Allow pet caretakers to accept/decline bookings - <A system that lets caretakers confirm or decline bookings that they receive>
    3. Create incident reports -  <A page that allows caretaeksr to create reports if there are any incidents with pets, so the it is documented>
    4. Reply to reviews - .<See reviews posted to them, and allow thm to add replies to reviews>
